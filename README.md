# Conceitos Básicos em C

Este repositório contém pequenos programas desenvolvidos durante aulas de programação em **C**.

O objetivo é praticar conceitos fundamentais da linguagem como:

* Tipos de dados
* Entrada e saída de dados
* Operações matemáticas
* Operadores de comparação
* Manipulação básica de variáveis

---

# Arquivos do projeto

### soma.c

Programa que solicita dois números ao usuário e exibe a soma.

### saldo_bancario.c

Demonstração de diferentes tipos de dados em C:

* `int`
* `float`
* `double`
* `char`
* `string (array de char)`

### operacoes_matematicas.c

Exemplo de operadores aritméticos e operadores de atribuição.

### inventario_de_estoque.c

Simulação simples de controle de estoque de produtos utilizando variáveis, comparações e cálculos.

---

# Compilação

Para compilar qualquer arquivo:

```bash
gcc nome_do_arquivo.c -o programa
```

Exemplo:

```bash
gcc soma.c -o soma
```

---

# Execução

```bash
./soma
```
